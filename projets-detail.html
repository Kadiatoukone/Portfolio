Projet 01

AppMusic

Description

AppMusic est une application web qui permet de rechercher des sons et de créer des playlists personnalisées. L'utilisateur peut parcourir un catalogue musical, écouter des extraits et organiser ses morceaux favoris.

Fonctionnalités

  • Recherche de musiques par titre, artiste ou genre
  • Création et gestion de playlists
  • Lecture d'extraits audio intégrée
  • Sauvegarde des préférences utilisateur

Compétences acquises

  • Mettre à disposition un service informatique
  • Répondre aux demandes d’assistance et d’évolution
  • Travailler en mode projet

Technologies

React Bootstrap JavaScript CSS

Projet 03

Dreameez

Description

Dreameez est un réseau social dédié au partage de rêves. Les utilisateurs peuvent publier leurs rêves, les commenter, les aimer et suivre d'autres rêveurs. L'interface encourage l'exploration de l'imaginaire collectif.

Fonctionnalités

  • Publication et partage de rêves
  • Système de likes et de commentaires
  • Suivi d'autres utilisateurs (follow/unfollow)
  • Fil d'actualité personnalisé
  • Catégorisation des rêves par tags

Compétences acquises

  • Travailler en mode projet
  • Répondre aux demandes d’assistance et d’évolution

Technologies

HTML CSS JavaScript

Projet 04

Librairie

Description

Librairie est une application web permettant de rechercher et d'enregistrer des livres via l'API Google Books. Les utilisateurs peuvent gérer leur bibliothèque personnelle, noter les livres lus et suivre leurs lectures.

Fonctionnalités

  • Recherche de livres via l'API Google Books
  • Ajout de livres à sa bibliothèque personnelle
  • Notation et avis sur les livres lus
  • Suivi de l'avancement de lecture
  • Export de la liste de lecture

Compétences acquises

  • Travailler en mode projet
  • Mettre à disposition des utilisateurs un service informatique

Technologies

HTML CSS JavaScript PHP Google API
