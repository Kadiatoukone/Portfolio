Projet 05

M2L_webapp

Description

Application web d'administration destinée au personnel de la Maison des Ligues de Lorraine (M2L), une structure régionale financée par le Conseil Régional mettant à disposition des ligues sportives des salles de réunion, amphithéâtres et complexes sportifs. L'outil remplace une gestion manuelle par téléphone et courrier, source d'erreurs et de doubles réservations.

Fonctionnalités

  • Dashboard synthétique des demandes, alertes et du calendrier global en temps réel
  • Validation ou refus des demandes de réservation des ligues
  • Gestion des salles : ajout et modification des caractéristiques (superficie, équipements, type de sport…)
  • Messagerie interne pour faciliter le dialogue entre la M2L et les ligues
  • Module de notation et commentaires sur les salles

Compétences acquises

  • Travailler en mode projet avec méthode Agile Scrum
  • Concevoir et modéliser une base de données relationnelle (MCD/MLD/MPD)
  • Développer une interface web responsive et sécurisée
  • Mettre en œuvre une architecture découplée (API REST + Frontend)
  • Appliquer les bonnes pratiques de sécurité (OWASP, RGPD, BCrypt)

Technologies

ReactJS Bootstrap CSS API REST MariaDB

Projet 06

M2L_mobile

Description

Application mobile destinée aux adhérents de ligues sportives de la Maison des Ligues de Lorraine (M2L). Elle leur permet de consulter les disponibilités et de réserver des salles de réunion, espaces de Team Building ou terrains d'entraînement 24h/24 et 7j/7, sans dépendre des horaires du secrétariat. L'objectif est de réduire de 70 % les appels entrants liés aux réservations.

Fonctionnalités

  • Authentification sécurisée avec MFA (code de vérification ou biométrie)
  • Recherche et consultation des salles disponibles par type et localisation
  • Réservation en ligne avec sélection de créneau et validation de la demande
  • Suivi de l'état des réservations (En attente / Validée)
  • Réception de confirmations et notifications par SMS ou e-mail
  • Modification ou annulation d'une réservation existante

Compétences acquises

  • Développer une application mobile cross-platform (iOS & Android)
  • Travailler en mode projet avec méthode Agile Scrum (Notion, Git)
  • Consommer une API REST depuis un client mobile
  • Mettre à disposition des utilisateurs un service informatique disponible en permanence
  • Sécuriser les accès et protéger les données personnelles (RGPD)

Technologies

React Native Expo API REST MariaDB
