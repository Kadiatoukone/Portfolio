Stage 1ère année

Yes I Am Free
Intégration web Front-end Responsive
2 mois Avril 2025-juin 2025
Contexte & Mission

Dans le cadre de ma première année de BTS SIO, j'ai réalisé un stage de 2 mois au sein de l'entreprise Yes I Am Free. Ma mission principale était la réalisation du site Free S-Cool à partir de maquettes fournies, en assurant l'intégration complète des pages en HTML et CSS avec un rendu responsive sur tous les supports.

Missions réalisées
  • Analyse et compréhension des maquettes graphiques fournies
  • Intégration fidèle des maquettes en HTML / CSS
  • Création de l'ensemble des pages du site Free S-Cool
  • Mise en place du responsive design (mobile, tablette, desktop)
  • Tests et validation du rendu sur différents navigateurs
  • Livraison et présentation du site au maître de stage
Compétences acquises
Intégration HTML/CSS depuis maquette
Responsive design (media queries)
Gestion de projet en autonomie
Respect des délais et livrables
Communication avec le client
Technologies : HTML5 CSS3

Stage 2ème année

NG Tech Consulting
API REST Back-end Full-stack
5 semaines février 2026-mai 2026
Contexte & Mission

Pour ma deuxième année de BTS SIO, j'ai de nouveau intégré Yes I Am Free pour un stage de 5 semaines centré sur le développement back-end. J'ai développé une API REST avec Symfony permettant la gestion de tâches, puis connecté cette API à une interface front-end en React, couvrant ainsi un périmètre full-stack.

Missions réalisées
  • Conception et architecture de l'API REST sous Symfony
  • Création et sécurisation des routes API (authentification JWT)
  • Développement des opérations CRUD pour la gestion de tâches
  • Connexion de l'API à une interface React (fetch / Axios)
  • Gestion des erreurs et des réponses HTTP
  • Tests des endpoints via Postman
  • Documentation de l'API et mise en production
Compétences acquises
Développement API REST avec Symfony
Architecture MVC et ORM Doctrine
Authentification JWT
Consommation d'API avec React
Tests API avec Postman
Gestion de base de données MySQL
Technologies : Symfony PHP React API REST Bootstrap MySQL
